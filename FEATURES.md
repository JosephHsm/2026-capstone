# 기능 상세 설명 (Feature Details)

## 📊 1. 통합 대시보드 (Dashboard)

### 핵심 지표 카드
- **충전 중 차량**: 전체 충전소의 현재 충전 차량 수 및 가동률
- **평균 ESS 잔량**: 모든 충전소의 배터리 평균 잔량
- **태양광 발전량**: 실시간 태양광 총 발전량
- **충전소 상태**: 정상/경고/오류 상태별 충전소 수

### 차트 및 시각화
- **24시간 전력 소비 현황**: Area Chart로 시간대별 전력 소비와 태양광 발전 표시
- **실시간 수요 예측**: Line Chart로 AI 예측 수요와 실제 소비 비교
- **충전소별 상태 테이블**: 모든 충전소의 실시간 상태 일람

### 부가 정보
- **기상 정보 위젯**: 기온, 습도, 운량, 일사량
- **AI 최적화 성과**: 비용 절감액과 보상 점수

### 데이터 갱신
- 5초마다 자동 리페치
- TanStack Query 캐싱으로 네트워크 최적화

---

## 🗺️ 2. 지도 뷰 (Map View)

### 카카오 맵 통합
- **지리적 시각화**: 서울 지역 충전소 위치 마커 표시
- **상태별 색상 코딩**:
  - 🟢 녹색: 정상 운영
  - 🟡 노란색: 경고 (90% 이상 사용률)
  - 🔴 빨간색: 오류 (시스템 이상)

### 인터랙티브 기능
- **마커 클릭**: 충전소 상세 정보 패널 표시
- **자동 줌**: 선택된 충전소로 지도 중심 이동
- **클러스터링**: 줌 아웃 시 여러 마커 그룹화 (향후 추가 예정)

### 상세 정보 패널
- 충전소 이름 및 ID
- 실시간 상태 (정상/경고/오류)
- 충전 현황 (현재 차량 / 최대 용량)
- ESS 잔량 (진행률 바)
- 태양광 발전량 및 계통 사용량

### 범례
- 상태별 마커 색상 설명
- 사용자 가이드

## ⚡ 3. 에너지 모니터링 (Energy Monitoring)

### 실시간 기상 정보
- **기온**: 현재 온도 (°C)
- **습도**: 상대 습도 (%)
- **운량**: 구름 양 (%)
- **일사량**: 태양 복사 에너지 (W/m²)

각 항목은 아이콘과 함께 카드 형식으로 표시

### 발전 현황
- **태양광 발전량**: 전체 충전소 합계 (kW)
- **계통전력 사용량**: 현재 전력망에서 공급받는 전력
- **재생에너지 비율**: 전체 에너지 중 태양광 비율

각 항목에 설비용량 대비 비율을 진행률 바로 표시

### 차트
1. **태양광 발전량 예측 vs 실제**
   - Area Chart
   - AI 예측값과 실제 발전량 비교
   - 24시간 데이터

2. **에너지원별 구성**
   - Horizontal Bar Chart
   - 태양광 / 계통전력 / ESS 방전
   - 실시간 비율

3. **24시간 에너지 생산·소비**
   - Line Chart
   - 소비량과 태양광 생산량 추이

4. **충전소별 태양광 발전 현황**
   - 진행률 바 리스트
   - 각 충전소의 실시간 발전량

### 환경 기여도
- **일일 CO₂ 저감량**: kg 단위, 소나무 그루 수 환산
- **일일 재생에너지 발전량**: kWh 단위, 가구 수 환산
- **연간 CO₂ 저감 예상**: ton 단위, 승용차 대수 환산

## 🤖 4. AI 최적화 (AI Optimization)

### 성과 지표 대시보드
1. **비용 절감액**
   - 월 누적 절감 금액
   - 진행률 바 (목표 대비)
   - 녹색 강조

2. **보상 점수**
   - 강화학습 보상 함수 평균값 (0~1)
   - 모델 성능 지표

3. **ESS 수명 페널티**
   - 과도한 충방전으로 인한 비용
   - 최소화 목표

4. **피크 초과 페널티**
   - 계약전력 초과 사용 비용
   - 최소화 목표

### 24시간 스케줄링 타임라인
- **Area Chart (누적)**:
  - ESS 충전 (파란색)
  - ESS 방전 (녹색)
  - 계통 구매 (주황색)
  - 태양광 발전 (연한 주황색)

### 스케줄 세부사항
- **야간 충전 (22:00-06:00)**: 저렴한 심야 전력 활용
- **피크 방전 (10:00-14:00)**: 비싼 전력요금 절감
- **태양광 활용 (09:00-17:00)**: 재생에너지 우선 사용
- **수요 반응 (실시간)**: 동적 부하 조정

### 성과 분석
1. **30일 보상 점수 추이**
   - Line Chart
   - 일별 보상 함수값
   - 학습 진행 상황 확인

2. **AI 의사결정 분포**
   - Bar Chart
   - ESS 충전/방전, 계통 구매, 대기
   - 각 행동의 선택 빈도

### 모델 정보
- **모델 구조**: Deep Q-Network (DQN), 4층 신경망
- **보상 함수**: 비용 절감, ESS 수명, 피크 초과, 재생에너지 가중치
- **학습 파라미터**: 학습률, 할인율, 탐험율, 배치 크기

## 📈 5. 통계 및 리포트 (Statistics)

### 필터 및 검색
- **기간 선택**: 일별 / 주별 / 월별 / 연별
- **충전소 선택**: 전체 또는 특정 충전소
- **리포트 다운로드**: Excel/PDF 내보내기 (향후 구현)

### 핵심 통계
- **총 절감액**: 6개월 누적
- **평균 절감률**: AI 도입 효과
- **이번 달 전력 사용**: 전월 대비 변화율
- **이번 달 전력 요금**: 전월 대비 절감액

### 탭 1: 비용 분석
1. **AI 도입 전/후 비용 비교**
   - Bar Chart (그룹)
   - 월별 비용 비교
   - 절감액 및 절감률 표시

2. **비용 구성 내역**
   - Pie Chart
   - 기본요금 / 전력량요금 / 피크요금 / 기타
   - 비율 및 금액

3. **월별 비용 추이**
   - Line Chart
   - 12개월 전력 비용 변화

### 탭 2: 전력 사용
1. **일별 전력 소비 추이**
   - Area Chart
   - 30일간 소비량 및 태양광 발전량

2. **시간대별 전력 사용 패턴**
   - Bar Chart
   - 24시간 소비/발전 패턴

### 탭 3: 효율성
1. **월별 운영 효율성**
   - Line Chart
   - 12개월 효율성 지표

2. **효율성 지표 카드**
   - **재생에너지 사용률**: 전체 에너지 중 태양광 비율
   - **ESS 활용률**: 배터리 충방전 효율
   - **피크 회피율**: 피크 시간대 전력 절감률

## 🎛️ 6. 관리자 콘솔 (Admin Console)

### 시스템 상태 모니터링
- **시스템 상태**: 정상/경고/오류 표시
- **AI 자동 제어**: 활성화/비활성화 토글
- **알림 및 경고**: 현재 경고 사항 수

### AI 제어 명령 이력
- **테이블 형식**:
  - 시간 (분:초 단위)
  - 충전소명
  - 제어 명령 (ESS 충전/방전, 태양광 우선 등)
  - 전력 (kW)
  - 상태 (실행됨/실패)

### 수동 제어 패널
**주의**: AI 자동 제어가 비활성화된 경우에만 사용 가능

#### 충전소 선택
- 버튼 그리드로 충전소 선택
- 선택된 충전소 강조 표시

#### 전력 제한 설정
- 슬라이더로 0-100% 설정
- 계약전력 대비 사용 한도

#### 제어 버튼
1. **ESS 충전**: 배터리 충전 시작
2. **ESS 방전**: 배터리 방전 시작
3. **일시 정지**: 모든 작업 일시 정지 (확인 다이얼로그)
4. **긴급 정지**: 즉시 모든 작업 중단 (확인 다이얼로그)

### 충전소별 제어 테이블
- **상태 표시**: 정상/경고/오류 배지
- **ESS 잔량**: 진행률 바 + 백분율
- **현재 전력**: 실시간 소비량
- **제어 버튼**:
  - 재시작 (RefreshCw 아이콘)
  - 긴급 정지 (Power 아이콘, 확인 다이얼로그)

### 알림 시스템
- **Toast 알림**: 제어 명령 실행 결과
- **색상 구분**:
  - 녹색: 성공
  - 노란색: 경고
  - 빨간색: 오류/긴급

## 📊 데이터 구조

### ChargingStation
```typescript
{
  id: string;              // 충전소 ID
  name: string;            // 충전소명
  lat: number;             // 위도
  lng: number;             // 경도
  status: string;          // 상태
  currentVehicles: number; // 현재 충전 차량
  maxCapacity: number;     // 최대 용량
  batteryLevel: number;    // ESS 잔량 (%)
  solarGeneration: number; // 태양광 발전 (kW)
  gridConsumption: number; // 계통 사용 (kW)
}
```

### AIMetrics
```typescript
{
  costSavings: number;        // 비용 절감액
  essLifetimePenalty: number; // ESS 수명 페널티
  peakPowerPenalty: number;   // 피크 초과 페널티
  rewardScore: number;        // 보상 점수 (0-1)
}
```

### PowerSchedule
```typescript
{
  time: string;            // 시간 (HH:MM)
  charging: number;        // ESS 충전 (kW)
  discharging: number;     // ESS 방전 (kW)
  gridImport: number;      // 계통 구매 (kW)
  solarGeneration: number; // 태양광 발전 (kW)
}
```
