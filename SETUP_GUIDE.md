# 빠른 시작 가이드 (Quick Start Guide)

## 📋 필수 요구사항

- Node.js 18 이상
- npm 또는 pnpm 패키지 매니저
- 카카오 개발자 계정 (지도 기능 사용 시)

## 🚀 5분 안에 시작하기

### 1단계: 프로젝트 클론 및 의존성 설치

```bash
# 의존성 설치
pnpm install
```

### 2단계: 개발 서버 실행

```bash
pnpm dev
```

서버가 실행되면 브라우저에서 `http://localhost:5173`을 엽니다.

✅ **지금 바로 대부분의 기능을 사용할 수 있습니다!**

- ✅ 통합 대시보드
- ✅ 에너지 모니터링
- ✅ AI 최적화 지표
- ✅ 통계 및 리포트
- ✅ 관리자 콘솔
- ⚠️ 지도 뷰 (API 키 필요)

## 🗺️ 카카오 맵 API 설정 (선택 사항)

지도 뷰 기능을 사용하려면 카카오 맵 API 키가 필요합니다.

### 카카오 개발자 계정 없는 경우

**걱정하지 마세요!** 지도 기능 없이도 다른 모든 기능이 정상적으로 작동합니다.

### 카카오 맵 API 키 설정하기

#### 1. API 키 발급

1. [Kakao Developers](https://developers.kakao.com/) 방문
2. 로그인 후 "내 애플리케이션" 클릭
3. "애플리케이션 추가하기" 클릭
4. 앱 이름 입력 및 생성
5. 생성된 앱의 "JavaScript 키" 복사

#### 2. 플랫폼 등록

1. 앱 설정 > 플랫폼 > "Web 플랫폼 등록" 클릭
2. 사이트 도메인 입력:
   - 개발: `http://localhost:5173`
   - 프로덕션: 실제 도메인
3. 저장

#### 3. 코드에 API 키 적용

파일: `/src/app/pages/MapView.tsx`

```typescript
// 38번째 줄 근처에서 찾기
script.src = `//dapi.kakao.com/v2/maps/sdk.js?appkey=YOUR_KAKAO_MAP_API_KEY&autoload=false`;

// 다음과 같이 수정
script.src = `//dapi.kakao.com/v2/maps/sdk.js?appkey=여기에_발급받은_키_붙여넣기&autoload=false`;
```

#### 4. 개발 서버 재시작

```bash
# Ctrl+C로 서버 중단 후
pnpm dev
```

이제 지도 뷰 페이지에서 충전소 위치를 확인할 수 있습니다! 🎉

- 재생에너지 비율
- 환경 기여도

- 24시간 ESS 충방전 스케줄
- AI 의사결정 분석
- 모델 파라미터 정보

### 5. 통계 및 리포트 (`/statistics`)
- AI 도입 전후 비용 비교
- 전력 사용 패턴 분석
- 운영 효율성 지표
- 기간별 필터링

### 6. 관리자 콘솔 (`/admin`)
- AI 자동 제어 토글
- 수동 제어 패널
- 충전소별 개별 제어
- 긴급 정지 기능

## 🎨 UI 특징 및 인터랙션

### 차트 인터랙션
- **호버**: 마우스를 차트 위에 올리면 상세 정보 표시
- **범례 클릭**: 특정 데이터 계열 숨기기/표시
- **반응형**: 화면 크기에 따라 자동 조정

### 실시간 업데이트
- 모든 데이터는 5초마다 자동 갱신
- 네트워크 상태에 따라 캐싱 활용
- 백그라운드 자동 리페치

### 알림 시스템
- 관리자 콘솔에서 제어 명령 실행 시 알림 표시
- 성공/경고/오류 상태별 색상 구분
- 화면 우측 상단에 표시

## 🔧 문제 해결

### Q: 개발 서버가 실행되지 않아요
```bash
# node_modules 삭제 후 재설치
rm -rf node_modules
pnpm install
pnpm dev
```

### Q: 지도가 표시되지 않아요
- 카카오 API 키를 확인하세요
- 플랫폼 도메인이 등록되어 있는지 확인하세요
- 브라우저 콘솔에서 에러 메시지를 확인하세요

### Q: 차트가 이상하게 보여요
- 브라우저 캐시를 지우고 새로고침하세요 (Ctrl+Shift+R)
- 브라우저 창 크기를 조절해보세요

### Q: 데이터가 실시간으로 바뀌지 않아요
- Mock 데이터는 랜덤으로 생성되므로 변화가 미미할 수 있습니다
- 5초마다 자동 갱신되므로 잠시 기다려보세요
